# Easy Touch （易触）
Easy Touch is a simple Material Design NFC read and write tool for Android

Easy Touch （易触）是Android上的一款使用Material Design设计的NFC读写工具

功能
=========================
支持写入，读取。

可以将Easy Scan中的OCR扫描结果写入进NFC贴纸，然后下次再触碰贴纸即可自动跳转到Easy Touch。
